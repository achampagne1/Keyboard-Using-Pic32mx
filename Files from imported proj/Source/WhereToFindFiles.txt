The files "mouse.c" and "usb_descriptors.c" are local to this project.

Up-to-date versions of the other .c files can copied from:
    "C:\{MICROCHIP_SOLUTIONS}\Microchip\USB\usb_device.c"
    "C:\{MICROCHIP_SOLUTIONS}\Microchip\USB\HID Device Driver\usb_function_hid.c"
    
where {MICROCHIP_SOLUTIONS} represents the directory, or path to the 
directory, that contains the Microchip Applications Library (MAL).
